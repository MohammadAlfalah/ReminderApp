ReminderApp – Android Reminder Application
A native Android application for managing personal reminders and tasks, built with Kotlin and XML.

Overview
ReminderApp is a mobile application developed to practice Android development with native Kotlin and XML-based UI design. The app allows users to create, view, and manage reminders with a clean and structured interface.

Tech Stack
Language: Kotlin

UI: XML Layouts (Android Views)

Platform: Android (Android Studio)

Version Control: Git

Features
Create and manage reminders

Structured XML-based UI with multiple screens

Navigation between activities/fragments

Basic local data handling with Kotlin logic

Purpose
Built as a personal Android development project to apply Kotlin programming skills and XML layout design in a real mobile application context.
